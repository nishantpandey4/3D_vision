F1 score graph for mesh

Loss function graph for mesh

F1 score graph for voxel grid

Loss function graph for voxel grid

F1 score graph for point_cloud

Loss function graph for point_cloud

As it evident in the graphs the point cloud have highest F1 score followed by mesh and then voxel grid. This could be because of the point cloud use highest number of n_points, while the voxel grid ground truth has low resolution and the mesh has regularization which effects its F1 score. We also know that the F1 score is inversely proportional to the training loss. Which is evident in the graphs as well.
